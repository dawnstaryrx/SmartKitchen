# SmartKitchen 🍳

SmartKitchen 是一个基于 FastAPI + Next.js 开发的智能厨房管理平台，旨在帮助用户管理家庭食材、学习烹饪知识、了解厨房工具，并结合 AI 能力提供智能推荐服务。

## 项目简介

在日常生活中，人们经常遇到以下问题：

- 冰箱里的食材还能保存多久？
- 某种蔬菜、水果应该如何存储？
- 食材应该如何处理和切配？
- 家里现有食材能做哪些菜？
- 某道菜应该如何制作？
- 厨房工具应该如何使用？

SmartKitchen 将食材知识库、菜谱系统、厨房工具百科与 AI 智能推荐相结合，为用户提供一站式厨房助手服务。

## 核心功能

### 1. 用户中心

- 用户注册
- 邮箱验证码登录
- JWT 身份认证
- 用户信息管理
- 权限控制

------

### 2. 食材管理系统

建立完整的厨房食材知识库。

#### 食材分类

支持多种食材分类：

- 蔬菜
- 水果
- 肉类
- 水产海鲜
- 米面粮油
- 调味料
- 豆制品
- 蛋奶制品
- 速冻食品
- 半成品食材
- 加工食品

#### 食材信息

每种食材包含：

- 基本介绍
- 营养成分
- 食用价值
- 适宜人群
- 禁忌搭配

#### 保存指南

- 常温保存
- 冷藏保存
- 冷冻保存
- 保存时长
- 变质判断方法

#### 食材处理

- 清洗方式
- 去皮方法
- 切配技巧
- 烹饪建议

#### 智能推荐

根据当前食材推荐：

- 可制作菜品
- 相似食材替代方案
- 推荐搭配食材

#### 自动购物清单

用户选择一周菜单：

```
红烧肉
宫保鸡丁
西红柿炒蛋
```

自动生成：

```
购物清单

五花肉 500g
鸡胸肉 300g
花生米 100g
鸡蛋 10个
西红柿 6个
```

这对家庭用户很有吸引力。

------

### 3. 菜谱管理系统

提供完整的菜谱知识库。

#### 菜谱信息

- 菜谱名称
- 菜品分类
- 菜品图片
- 难度等级
- 烹饪时间
- 口味标签

#### 制作步骤

- 所需食材
- 所需调料
- 制作流程
- 图文步骤
- 烹饪技巧

#### 菜谱搜索

支持：

- 菜名搜索
- 食材搜索
- 分类筛选
- 难度筛选

#### 智能菜谱推荐

根据用户已有食材：

- 推荐可制作菜谱
- 推荐缺少材料较少的菜谱
- 推荐营养搭配方案

------

### 4. 厨房工具百科

建立厨房工具知识库。

#### 厨房工具分类

- 刀具
- 锅具
- 灶具
- 电器设备
- 烘焙工具
- 储存工具

#### 工具信息

- 工具介绍
- 使用方法
- 使用场景
- 清洁保养
- 安全注意事项

------

### 5. AI 智能助手

结合大语言模型提供智能服务：

#### AI 食谱推荐

输入现有食材：

> 西红柿、鸡蛋、面条

自动生成：

- 推荐菜谱
- 制作步骤
- 营养分析

#### AI 厨房问答

例如：

- 土豆发芽还能吃吗？
- 牛肉怎么解冻最好？
- 西兰花如何保存？

#### AI 菜谱生成

根据用户需求生成个性化菜谱：

- 减脂餐
- 健身餐
- 儿童餐
- 家常菜

### 6. 库存管理

记录家里实际拥有的食材：

- 我的冰箱
- 我的储藏室
- 剩余数量
- 购买日期
- 保质期
- 即将过期提醒

例如：

> 鸡蛋（剩余 5 个，3 天后过期）

这是 SmartKitchen 最有价值的功能之一。

### 7. 菜单规划

- 今日菜单
- 一周菜单
- 营养搭配
- 购物清单自动生成

例如：

周菜单：

- 周一：西红柿炒蛋
- 周二：红烧肉
- 周三：宫保鸡丁

自动生成需要购买的食材列表。

## 项目目标

SmartKitchen 不仅是一个菜谱管理系统，更希望成为用户的智能厨房助手：

- 管理食材
- 学习烹饪
- 提升厨房效率
- 减少食材浪费
- 提供 AI 智能推荐

最终来构建一个集 **食材知识库 + 菜谱系统 + 厨房百科 + AI 助手** 于一体的智能厨房平台。
